AI動画台本作成アシスタント
取扱説明書（シンプル版）

--------------------------------------------------

■ 1. このAIでできること

このAIは、資料から動画台本を作るアシスタントです。

AIが行うこと

・資料理解
・動画構成作成
・台本作成
・テロップ作成

出力は動画編集用の台本表になります。


--------------------------------------------------

■ 2. 必要ファイル

AIに以下のファイルを読み込ませます。

video_script_system.yaml
module_dictionary.yaml
module_transition_map.yaml

役割

video_script_system.yaml
台本生成ルール

module_dictionary.yaml
動画構造の部品

module_transition_map.yaml
モジュールの流れ


--------------------------------------------------

■ 3. ChatGPTでの設定

1. ChatGPTを開く
2. 新しいGPTを作成
3. Knowledge に以下を追加

video_script_system.yaml
module_dictionary.yaml
module_transition_map.yaml

4. 保存

これで準備完了です。


--------------------------------------------------

■ 4. Geminiでの設定

1. Geminiを開く
2. Gems を作成
3. Files / Knowledge に以下を追加

video_script_system.yaml
module_dictionary.yaml
module_transition_map.yaml

4. 保存


--------------------------------------------------

■ 5. AIへの依頼

AIには次のように依頼します。

「この資料から動画台本を作ってください。

条件
・モジュール構造
・テロップ作成
・ナレーション作成
・表形式で出力」


資料は

・企画書
・文字起こし
・商品説明
・インタビュー

など何でも使えます。


--------------------------------------------------

■ 6. このAIの使い方

このAIは、最初から完成稿を確定するためのものではありません。
まず叩き台を書き出し、それを見ながら人が修正指示を出して仕上げていきます。

たとえば次のように依頼できます。

・ここをもっと短くしてください
・別案を3つ出してください
・このモジュールを前に移動してください
・この部分はもっとアイデアを出してください
・このCUTはナレーションなしにしてください

AIはその指示をもとに、自動で再構成・修正します。


--------------------------------------------------

■ 7. 出力形式

AIは以下の形式で台本を作ります。

module_no
structure_role
cut_no
visual_description
telop
narration
note

モジュール = 動画構造
カット = 編集単位
note = 補足メモ


--------------------------------------------------

■ 8. 配布セット

他の人に渡すときは以下をセットにします。

取扱説明書.txt
video_script_system.yaml
module_dictionary.yaml
module_transition_map.yaml

これで誰でも同じAIを使えます。
