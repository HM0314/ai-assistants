# AI Assistants

This repository contains small AI assistants used in real workflows.

このリポジトリには、実際の制作ワークフローで使用している  
小さなAIアシスタントをまとめています。

---

## Assistants / アシスタント

### Script Assistant

Generates modular video scripts from source materials.

資料からモジュール構造の動画台本を生成するAIアシスタント。

Location:

script-assistant/


---

### Poster / Flyer Assistant

Generates structured layouts for posters and flyers.

チラシやポスターの構成レイアウトを生成するAIアシスタント。

Location:

poster-flyer-assistant/
