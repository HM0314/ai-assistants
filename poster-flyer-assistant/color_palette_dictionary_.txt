color_palette_dictionary:
  dictionary_name: "color_palette_dictionary"
  version: "3.0.0"
  purpose: >
    ポスター・チラシ制作システムにおける配色プリセットを定義し、
    HTML preview や後段のデザイン生成で統一感のある見た目を再利用できるようにする。
    用途名ではなく、見え方・トーン・読みやすさの観点で palette を整理する。

  design_principles:
    - "palette は色そのものではなく、役割付きカラートークンとして定義する"
    - "palette は module や block とは独立した見た目レイヤーとして扱う"
    - "構図辞書と同様に、palette も用途より見え方を優先して整理する"
    - "CTA や強調表現には accent / highlight を限定的に使い、全体の統一感を崩さない"
    - "light / dark の背景方針と、文字のコントラストを明示する"

  palette_definition_policy:
    token_structure:
      - "primary"
      - "secondary"
      - "accent"
      - "background"
      - "surface"
      - "text_main"
      - "text_sub"
      - "border"
      - "highlight"
    relation_to_layout: "slot や座標を定義せず、preview / render に色トークンを渡す"
    relation_to_branding: "公開用は匿名 palette とし、必要に応じてローカルで実名ブランド版に差し替える"
    relation_to_copy: "copy の意味を変えず、強弱や読みやすさを補助する"
    relation_to_preview: "palette は HTML preview の統一感と完成度を高めるために使用する"
